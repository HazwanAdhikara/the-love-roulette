# The Love Roulette

The Love Roulette is a psychological horror-thriller visual novel built with the Ren'Py engine.

## Overview

The story follows the toxic and twisted relationship between Arthur Pendelton, a cold and analytical former detective, and Tabitha Vane, an explosive and traumatized pediatric nurse. Pushed to their absolute limits by a cruel world, they decide to settle their domestic dispute through a deadly, strategic game of Russian roulette.

## Project Notes

- Engine: Ren'Py
- Genre: Psychological horror-thriller visual novel
- Language: English

## Repository Hygiene

This repository ignores generated Ren'Py outputs, local save data, logs, and macOS system files so only the source project stays under version control.

## Getting Started

1. Open the project in Ren'Py.
2. Launch the game from the Ren'Py launcher.
3. Edit the script files under `game/` to continue development.
